# zzzz Mizu Icons for Sublime Text

Generated from Mizu Icons 2.10.1 for use with Cursor Dark and MSVC Light.

Install this folder into Sublime Text Packages and keep A File Icon installed.
The PNGs intentionally use A File Icon's `file_type_*.png` names so they replace
the active sidebar file icons with original-color Mizu icons.

For the most reliable override, copy `icons/multi/` into:

```text
~/Library/Application Support/Sublime Text/Packages/A File Icon/icons/multi/
```

The repo's top-level `install.zsh` does this automatically.
